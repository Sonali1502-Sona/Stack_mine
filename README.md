# 🎮 Stack Mine

A simple and addictive block-stacking arcade game built with HTML, CSS, and JavaScript.

## 🚀 Live Demo

https://sonali1502-sona.github.io/Stack_mine/

---

## 📌 Features

- Click, tap, or spacebar to drop blocks
- Score and best-score tracking
- Perfect-drop streak bonus
- Three speed modes (slow / normal / fast)
- Responsive UI
- Smooth canvas animations

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
